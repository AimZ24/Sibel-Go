# Sibel Go!
_Asisten Siswa Dalam Pembelajaran Berbasis Web_

## Deskripsi
Sibel Go! adalah aplikasi berbasis web yang berfungsi sebagai asisten untuk membantu siswa dalam proses pembelajaran. Dengan Sibel Go, siswa dapat mengakses berbagai fitur yang mempermudah belajar secara digital dan terstruktur.

## Fitur Utama
- **Antarmuka Web Sederhana**: Memudahkan akses dan penggunaan bagi siswa.
- **Asisten Pembelajaran**: Mendukung aktivitas belajar siswa dengan berbagai tools yang terintegrasi.
- **Lisensi MIT**: Sibel Go dirilis dengan lisensi MIT, sehingga dapat digunakan dan dikembangkan secara bebas.

## Library

- Bootstrap 5 (via CDN)
- Font Awesome (via CDN)

## Instalasi & Penggunaan
1. Clone repository ini:  
   ```bash
   git clone https://github.com/AimZ24/Sibel-Go.git
   ```
2. Buka folder hasil clone dan jalankan file `index.html` (atau file utama lainnya jika tersedia) di browser.
